# Antigravity – IMBA Frontier Technologies

This repository contains the group and individual work for the Antigravity project in the IMBA Frontier Technologies course (Vision 1 and Vision 2).  
It is the single home for our brief, research, slides, and individual reports.

- `docs/` – assignment briefs, clarifications, and project timeline.  
- `group-presentation/` – latest group decks and speaking scripts for Vision 1 and Vision 2.  
- `research/` – structured research notes, sources, and analysis by vision.  
- `individual/` – personal folders for each team member’s outline, drafts, and notes.  
- `data/` – non-sensitive external datasets and processed tables (if used).  
- `tools/` – AI prompts and helper scripts used for research and automation.

The latest group slides live in `group-presentation/vision1/slides/` (and `vision2/` if applicable); older versions go into `slides/old/`.
