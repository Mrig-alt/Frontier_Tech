# Antigravity – IMBA Frontier Technologies

This repository is the single home for the IMBA Frontier Technologies group and individual project ("Antigravity"). It contains the project brief, research notes, group presentation slides, and individual report work.

## Folder Structure

- `docs/` – Assignment briefs (individual + group), assumptions, and project timeline.
- `group-presentation/` – Group deck and speaking scripts.
  - `vision1/slides/` – Latest Vision 1 slides (current deck only; older versions go in `slides/old/`)
  - `vision2/slides/` – Vision 2 slides when applicable.
- `research/` – Structured research notes and sources.
  - `vision1/` – Brain mapping, IBM Quantum, government regulation notes.
  - `vision2/` – Agentic AI biology, space organoids notes.
- `individual/` – Personal workspaces for each team member (outline, drafts, data notes).
- `data/` – Non-sensitive external and processed data (if used), with a readme describing source + date.
- `tools/` – AI prompts and helper scripts used for research and automation.

## Workflow

- **main** – Clean, presentable state only (final decks, final exports, stable docs).
- **dev** – Work in progress.
- Personal branches: `<name>/feature` (optional).

Use **Issues** to track decisions: e.g., "Lock tech stack for Vision 2", "First draft of group deck", "Everyone upload individual outline".

## Getting Started

The latest group slides live in `group-presentation/vision1/slides/`. Team members should clone the repo, work on `dev` or a personal branch, and open a pull request when ready to merge into `main`.
